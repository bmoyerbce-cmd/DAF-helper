Corrective Action PDF Helper

Fill the existing DAF and Employee Counseling Record templates from one webpage, keep the original PDF layouts intact, save reusable templates, and manage dropdowns or field behavior from the Admin tab.

Quick preview

What will be filled

This preview shows the values that will be written into the currently selected PDF. Employee comments and signature lines stay untouched unless you later change those fields in Admin.

Tip: text boxes and text areas already use browser spellcheck. The Grammar Review button can also scan long narrative fields for grammar suggestions whenever you are online.

Reusable templates

Save frequently used wording so supervisors can prefill a form with one click before making any final edits.

Saved template library

Admin controls

Update dropdown options, swap a field between freeform and prefilled/static text, or change default values. Settings stay in the browser for this tool using local storage.

Field types

Supported field behavior

TextShort single-line entry such as names or dates.
TextareaLonger writeups such as reasons, policy notes, or corrective action language.
SelectDropdown list such as supervisor or warning level.
StaticAlways prefilled and read-only on the supervisor form.
HiddenNot shown in the supervisor form, but a default value can still be written into the PDF.
If you later want to turn a freeform field into a prefilled value, switch it to Static and set the default text in Admin.
